# Familiars-EDA
Business Intelligence dashboard using Microsoft Power BI Desktop with DAX, data modeling, KPI tracking, and time series analysis to improve profitability and sales performance insights.
This project focuses on building an interactive Retail Sales Performance Dashboard using Microsoft Power BI Desktop to analyze sales trends, profitability, customer behavior, and regional performance.

The dashboard helps identify top-performing products, low-margin categories, customer buying patterns, and monthly sales trends using KPI tracking, DAX measures, data modeling, and time series analysis.

The goal of this project is to support data-driven business decision-making by improving pricing strategies, inventory optimization, customer retention, and overall profitability.

This project demonstrates practical skills in Business Intelligence, Data Analytics, Dashboard Development, and Data Storytelling.
